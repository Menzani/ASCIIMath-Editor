Why is Firefox required in order to use this app?

This app is built on top of ASCIIMath, a powerful tool that can convert mathematical objects written in plain text and using an intuitive syntax to high-quality mathematical content that is rendered directly in your browser.

ASCIIMath works with the MathML specification to achieve beautiful results efficiently. However, Firefox is the only browser that supports the MathML standard at this time. As soon as other browsers feature a decent implementation of the standard they will be supported, too.

What would it take to support all browsers anyway?

Supporting all browsers would require the rendering to be implemented from scratch. However, doing so would be less-than-optimal for various reasons, mostly because it would severely impact performance. By leveraging a native technology like MathML, you get better results in less time, and you are free to enjoy an optimized experience that is crafted just for your browser.
