This app is designed to run on Firefox and likely won't support other browsers. Find out why

ASCIIMath Editor relies on Firefox's near-perfect MathML engine to perform the rendering, so you get better results in less time.
ASCIIMath Editor also leverages Firefox's high-performance animation rendering engine to delight the user with unintrusive, rich animations that are both fun and useful.
